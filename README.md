# Frontend Mentor - FAQ Accordion

A responsive FAQ accordion built as part of a Frontend Mentor challenge.

---

## Overview

This project is a simple FAQ accordion interface where users can expand and collapse questions. It is built with semantic HTML and CSS, focusing on accessibility and responsive design.

---

## Features

- Toggle FAQ answers using `<details>` and `<summary>`
- Keyboard accessible (no JavaScript required)
- Responsive layout for mobile and desktop
- Hover and focus states for better UX
- Clean and semantic HTML structure
