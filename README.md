Frontend Mentor - FAQ Accordion

This is my solution to the FAQ accordion challenge
on Frontend Mentor.

Overview

A responsive FAQ accordion built using semantic HTML and CSS. Users can expand and collapse questions, and the layout adapts to different screen sizes.

Features
Toggle FAQ answers (accordion behavior)
Keyboard accessible using <details> and <summary>
Responsive design (mobile + desktop)
Hover and focus states
Accessible structure using semantic HTML
Built With
HTML5
CSS3 (Flexbox + media queries)
Mobile-first workflow
What I Learned
How to use <details> and <summary> for accessible accordions
How CSS pseudo-elements (::after) work as layout elements
Handling image paths for GitHub Pages deployment
Fixing layout shift caused by scrollbars
Basics of accessibility (ARIA, screen readers)
